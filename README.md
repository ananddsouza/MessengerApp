# MessengerApp

A Simple Messenger app based on ReactJS.

## Available Scripts

To install packages on backend in project directory --> backend, run 

### `npm install`

To install packages on client in project directory --> react-app, run 

### `npm install`


In the project directory --> backend, you can run:

Runs the Server at port 9002.

In the project directory --> react-app, you can run:

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
